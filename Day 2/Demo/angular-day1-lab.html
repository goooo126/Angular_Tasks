Day 1 – Lab Assignment

Angular Foundations & Core Concepts

Project: TechStore

In this lab, you will build a small TechStore application from scratch. Every question builds on the previous one — by the end, you'll have a working store page with a header, navigation, product listing, sidebar, and footer.

Question 1

Project Setup & App Structure

CLI
Goal: Create a new Angular project called tech-store and generate the core components.

Steps

  1. Create a new Angular project: 
    ng new tech-store --no-standalone
  2. Navigate into the project folder and generate the following 4 components:
    • header
    • navbar
    • products
    • footer
    ng g c header
ng g c navbar
ng g c products
ng g c footer
  3. Open app.component.html, remove all default content, and add your components in this order: 
    <app-header></app-header>
<app-navbar></app-navbar>
<app-products></app-products>
<app-footer></app-footer>
  4. Run ng serve and verify all 4 components render on the page.

Expected Component Tree

AppComponent (Root)
  ├── HeaderComponent
  ├── NavbarComponent
  ├── ProductsComponent
  └── FooterComponent
Question 2

Create the Store Models

Models TypeScript
Goal: Define the data models that the store will use — a Product interface and a Category enum.

Steps

  1. Create a folder src/app/models/.
  2. Inside it, create a file product.model.ts with an interface called Product that has:
    PropertyType
    idnumber
    namestring
    pricenumber
    imageUrlstring
    inStockboolean
  3. In the same file (or a separate category.model.ts), create an enum called Category with values:
    • Phones
    • Laptops
    • Accessories
    export enum Category {
  Phones = 'Phones',
  Laptops = 'Laptops',
  Accessories = 'Accessories'
}
  4. Bonus Add a category property of type Category to the Product interface.
Hint: An interface defines the shape of an object. An enum defines a set of named constants. Both are TypeScript features — they don't exist at runtime.
Question 3

Build the Header Component

HTML TypeScript
Goal: The header should display the store name, a logo image, and a welcome message — all using interpolation and property binding.

In header.component.ts

  1. Add a property storeName of type string with value 'TechStore'.
  2. Add a property logoUrl of type string with value 'logo.png'.
  3. Add a property customerName of type string with your name.

In header.component.html

  1. Display the storeName inside an <h1> using interpolation {{ }}.
  2. Display the logo using <img> with property binding [src]="logoUrl".
  3. Display a welcome message: "Welcome, {{ customerName }}!"
Note: Place your images in the public/ folder (next to src/). Files in public/ are served at the root URL — so public/logo.png is accessed as 'logo.png' in your code.
Question 4

Build the Navbar Component

HTML TypeScript
Goal: Create a navigation bar that displays links from an array using *ngFor.

In navbar.component.ts

  1. Add a property navLinks — an array of objects, each with label (string) and url (string): 
    navLinks = [
  { label: 'Home', url: '#' },
  { label: 'Products', url: '#products' },
  { label: 'About', url: '#about' },
  { label: 'Contact', url: '#contact' }
];

In navbar.component.html

  1. Create a <nav> element.
  2. Inside it, use *ngFor to loop over navLinks and render an <a> tag for each link.
  3. Use interpolation for the link text: {{ link.label }}
  4. Use property binding for the href: [href]="link.url"
Hint: This is your first use of *ngFor. The syntax is: *ngFor="let link of navLinks"
Question 5

Build the Products Component

HTML TypeScript Models
Goal: Display a list of products using the Product interface you created in Q2, with *ngFor, interpolation, and property binding.

In products.component.ts

  1. Import the Product interface from ../models/product.model.
  2. Add a property products of type Product[] and initialize it with at least 4 products: 
    products: Product[] = [
  { id: 1, name: 'iPhone 15',       price: 999,  imageUrl: 'phone.png',  inStock: true  },
  { id: 2, name: 'MacBook Pro',     price: 1999, imageUrl: 'laptop.png', inStock: true  },
  { id: 3, name: 'AirPods Pro',     price: 249,  imageUrl: 'airpods.png',inStock: false },
  { id: 4, name: 'Samsung Galaxy',  price: 899,  imageUrl: 'galaxy.png', inStock: true  }
];
  3. Add a property storeName: string = 'TechStore'.

In products.component.html

  1. Add an <h2> that displays: {{ storeName }} — Our Products
  2. Use *ngFor to loop over the products array and render a card for each product containing:
    • An <img> with [src]="product.imageUrl"
    • An <h3> with {{ product.name }}
    • A <p> showing the price: Price: ${{ product.price }}
    • A <span> showing stock status using a ternary expression: 
      {{ product.inStock ? 'In Stock ✅' : 'Out of Stock ❌' }}
  3. Use property binding to conditionally disable a "Buy" button when the product is out of stock: 
    <button [disabled]="!product.inStock">Buy Now</button>
Note: Place product images in the public/ folder (e.g., public/phone.png). They are accessed directly as 'phone.png' in your code — no path prefix needed.
Question 6

Add a Sidebar Component

CLI HTML TypeScript
Goal: Add a sidebar that displays product categories using the Category enum, and update the page layout to show the sidebar next to the products.

Steps

  1. Generate a new component: 
    ng g c sidebar
  2. In sidebar.component.ts:
    • Import the Category enum from your models.
    • Create a property categories that holds all enum values: 
      categories = [Category.Phones, Category.Laptops, Category.Accessories];
  3. In sidebar.component.html:
    • Add a heading: <h3>Categories</h3>
    • Use *ngFor to render each category as a list item.
  4. Update app.component.html to place the sidebar next to the products section: 
    <app-header></app-header>
<app-navbar></app-navbar>

<div class="main-content">
  <app-sidebar></app-sidebar>
  <app-products></app-products>
</div>

<app-footer></app-footer>
  5. In app.component.css, add a simple layout: 
    .main-content {
  display: flex;
}
app-sidebar {
  width: 200px;
}
app-products {
  flex: 1;
}

Question 7

Build the Footer Component

HTML TypeScript
Goal: Create a footer that displays the store name and current year using interpolation and expressions.

In footer.component.ts

  1. Add a property storeName: string = 'TechStore'.
  2. Add a property currentYear that holds the current year: 
    currentYear = new Date().getFullYear();

In footer.component.html

  1. Display: © {{ currentYear }} {{ storeName }}. All rights reserved.
Bonus

Add Bootstrap & Style the App

Bonus CLI
Goal: Install Bootstrap and apply styling to make the store look polished.

Steps

  1. Install Bootstrap: 
    npm install bootstrap
  2. Add Bootstrap CSS to angular.json under styles: 
    "styles": [
  "node_modules/bootstrap/dist/css/bootstrap.min.css",
  "src/styles.css"
]
  3. Apply Bootstrap classes to your components:
    • Navbar: Use navbar, navbar-expand-lg, navbar-dark, bg-dark classes.
    • Products: Use card classes for product cards, laid out in a row with col-md-4.
    • Footer: Use bg-dark text-white text-center p-3.
  4. Use property binding for dynamic styling on product cards: 
    <!-- Grey out out-of-stock products -->
<div [class.opacity-50]="!product.inStock">...</div>
