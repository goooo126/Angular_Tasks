📋 Requirements

1 Create Components

Generate three components using the Angular CLI:

  • NavbarComponent — displays the app name and navigation links
  • ProductsComponent — the main component (all logic lives here)
  • FooterComponent — displays copyright with the current year

Use them in app.component.html.

Components Interpolation

2 Create Interfaces

Create ICategory and IProduct interfaces in a models/ folder. Use the sample data above.

3 Category Dropdown with Two-Way Binding

Add a <select> dropdown that lists all categories. When the user selects a category, filter and display only the products that belong to that category in the table.

  • Use [(ngModel)] to bind the selected category ID to a component property
  • Don't forget to import FormsModule in app.module.ts
  • Create a getter or method that returns filtered products based on the selected category

Two-Way Binding Event Binding

4 Display Products in a Table

Use *ngFor to loop through the filtered products and display them in a table with columns:

  • # — Row number (use let i = index)
  • Product Name
  • Category
  • Price
  • Quantity
  • Action (Buy button)

*ngFor Interpolation

5 Hide Products with Quantity = 0

Products with quantity === 0 should be hidden from the table. Use the *ngIf directive on the table row.

*ngIf

6 Conditional Row Styling

Change the row background color based on the product's quantity:

  • Quantity = 1 → Background color: yellow (last item warning)
  • Quantity > 1 → Default background (no special color)

Use [ngClass] or [ngStyle] or [style.background] on the <tr> element.

Class Binding Style Binding ngClass

7 Buy Button

Add a Buy button on each product row. When clicked:

  • Decrease the product's quantity by 1
  • When quantity reaches 0, the row should disappear (because of requirement 5)
  • When quantity reaches 1, the row should turn yellow (because of requirement 6)
  • Disable the button when quantity = 0

Event Binding Property Binding

8 Pipes — Format the Display

Use Angular's built-in pipes to format data in the template:

  • Product name → Display in UPPERCASE using the uppercase pipe
  • Price → Format as currency using currency:'USD' pipe
    Example: 1299 → $1,299.00
  • Date → Format using date:'mediumDate' pipe
    Example: 2026-03-17 → Mar 17, 2026
  • Footer → Display today's date using the date pipe
  • Products count → Show "Showing X products" in the header subtitle using interpolation

uppercase currency date
